# Military Girls Hostel Web Application

This project is a web-based management system for the Military Girls Hostel. It is built using React.js and Tailwind CSS for the frontend, and Firebase for the backend.

## Overview

The web application helps manage hostel activities such as student registration, outpass requests, complaints, visitor appointments, and administrative controls. It includes separate interfaces for residents and the administrator.

## Features

### Visitors
- View home page 
- Fill the admission application form
- View hostel photo gallery 
- Access contact info
- Read hostel rule book

### Resident Panel
- Login and Registration using Firebase Authentication
- Profile Management
- Outpass Request Form
- Complaint Submission
- Real-time status updates

### Admin Panel
- Admin Authentication
- Approve or reject student registrations
- View and mark outpass requests as seen
- Manage complaints
- Post and update hostel notices
- View visitor application forms
- Real-time Firestore updates for all collections

## Tech Stack
- React.js (Vite)
- Firebase Firestore
- Firebase Authentication
- Tailwind CSS
- React Hot Toast (for notifications)
